## Functions

There are two types of functions, static functions and dynamic functions.

### Static Functions

Static functions are executed at build time. They are optimized for performance and bundle size.

For more information, please refer to the [static functions documentation](./functions/static/README.md).

### Dynamic Functions

Dynamic functions are executed at run time. They are intended for cases that require dynamic behavior. But they must not be used inside static functions.

For more information, please refer to the [dynamic functions documentation](./functions/dynamic/README.md).
